## Project Overview

This project demonstrates the design and implementation of a secure departmental network using Cisco Packet Tracer. Separate VLANs were created for multiple departments to improve security, reduce broadcast traffic, and organize network resources. Inter-VLAN communication was implemented using router-on-a-stick with IEEE 802.1Q trunking, allowing devices in different VLANs to communicate securely while maintaining network segmentation.

---

## Objectives

- Design a departmental network
- Create multiple VLANs
- Assign switch ports to VLANs
- Configure IEEE 802.1Q trunking
- Configure Router-on-a-Stick
- Enable Inter-VLAN routing
- Verify connectivity between VLANs
- Troubleshoot VLAN and routing issues
